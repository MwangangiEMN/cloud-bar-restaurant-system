# Cloud-Based Bar & Restaurant Management System

## Overview
This project is a scalable cloud-based hospitality platform designed to modernise how users access food, drinks, and accommodation services through a unified system.

The platform enables customers to browse menus, place orders, track deliveries in real time, and book rooms based on availability. It also provides vendors with a dedicated dashboard to manage services, process orders, and monitor performance.

The system is designed with scalability, performance, and accessibility in mind, making it suitable for deployment across multiple regions and capable of supporting high user traffic.

## System Architecture

The system follows a **client-server architecture**:

1. Users interact with the system through a web-based interface  
2. Requests are sent to the backend server (Django application)  
3. The backend processes logic and communicates with the database  
4. Data is retrieved or updated in the MySQL database  
5. Responses are sent back to the user interface  

The architecture is designed to support multiple users simultaneously and ensure efficient data handling.
